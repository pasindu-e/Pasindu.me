# 💖 Romantic Surprise Website for Nikini 💖

## How to Add Your Photo

To personalize the surprise with your beautiful photo together:

1. **Get your photo ready:**
   - Choose a beautiful photo of you and Nikini together
   - Crop it to be square (1:1 ratio) for best results
   - Name it `your-photo.jpg` or `your-photo.png`

2. **Add the photo to your website:**
   - Copy your photo file to the same folder as `index.html`
   - Rename your photo file to: `your-photo.jpg`
   
3. **If you have a different file name or format:**
   - Open `index.html` in a text editor
   - Find this line: `<img src="your-photo.jpg" alt="Our Beautiful Moment" class="couple-photo"`
   - Change `your-photo.jpg` to your actual file name (e.g., `our-selfie.png`)

## What happens if no photo is added?
- Don't worry! There's a beautiful placeholder that will show instead
- The website will still work perfectly with all the romantic animations
